What is 2048 game?

2048 is a single-player sliding tile puzzle video game created by Gabriele Cirulli on 2014, this the game.

2048 is played with numbered tiles that slide when a player moves them using the four arrow keys.

Tiles with matching number values can be merged into a single tile, which receives the values' sum.

Every turn, a new tile randomly appears in an empty spot on the board with a value of either 2 or 4.

To win, get a 2048 tile.
